# FlowTasks Mobile (Demo)

A premium, mobile-first task manager demo built as a “product-grade” portfolio project.
Focus: clean UX, modular architecture, offline persistence, and shareable deep links.

## Key Features

- Mobile-first UI (app-like layout)
- Tasks list with status (done/active)
- Tags and quick filtering
- Add Task flow via modal
- Validation and clean form UX
- Offline persistence (localStorage)
- Deep links (e.g. `?tag=Work`) for shareable filtered views
- Keyboard UX: ESC closes modal

---

## Tech Stack

- React + Vite
- Context + Reducer (state management)
- localStorage persistence
- Minimal routing via URLSearchParams (deep links)
- CSS (premium glossy UI)
